📖 Guide de Prise en Main

Apprenez à dompter l'infrastructure PH FIRE AFRICA.

🤝 1. Le Réseau Social

Utilisez le Fil d'Actualité pour partager vos avancées. Chaque post peut contenir du texte, des images ou des vidéos techniques. Cliquez sur + Suivre pour bâtir votre réseau de collaborateurs.

📚 2. L'Académie d'Élite

Allez dans la section Éducation. Choisissez un domaine (ex: Programmation) et suivez les leçons dans l'ordre.

⚠️ Discipline : Vous devez répondre au Quiz ou valider l'exercice de code pour prouver votre maîtrise avant de passer à la suite.

💰 3. Extraire de la Valeur (La Mine)

PH FIRE récompense votre temps de cerveau.

  • Apprenants : Tant que vous lisez une leçon, votre Wallet augmente.
  • Professeurs : Vous gagnez des commissions sur chaque élève qui étudie votre cours.
Consultez votre Portefeuille pour voir votre richesse monter en temps réel.

🧠 4. L'IA Mwalimu

Si vous êtes bloqué sur un code Python ou un concept, cliquez sur l'icône 🧠. Mwalimu vous guidera sans donner la solution brute, pour forger votre intelligence.
